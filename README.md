# Voltai Sales OS

A local-first sales & marketing command center for Voltai — AI-prioritized
to-dos, an account hub (Attio · Notion · Granola · Slack), outreach, and a
LinkedIn-style relationship-map org chart. Native Mac app.

## ⬇️ Download

### **[Download Voltai Sales OS (latest)](https://github.com/voltavi/voltai-sales-os-releases/releases/latest/download/VoltaiSalesOS.zip)**

Or browse every version on the **[Releases page](https://github.com/voltavi/voltai-sales-os-releases/releases)**.

**Requirements:** Apple Silicon Mac (M1 or newer) · macOS 14 (Sonoma) or later.

**Install:**
1. Download and unzip.
2. Drag **Voltai Sales OS.app** into your **Applications** folder.
3. Open it — it's notarized by Apple, so it launches with no security warning.

The app **updates itself**: you'll be offered new versions automatically, or
check anytime via **Settings → Updates → Check for Updates**.

_This repository hosts the public releases and the Sparkle update feed

repo; releases are published with `mac/release.sh`._
